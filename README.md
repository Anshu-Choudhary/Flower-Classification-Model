# 🌸 Flower Classification using CNN

A deep learning project that classifies flower species from images using a Convolutional Neural Network (CNN), deployed as an interactive web app using Streamlit.

---

## 🎯 Project Overview

This project uses a CNN model built with TensorFlow and Keras to identify different types of flowers from uploaded images. The model was trained with image augmentation techniques to improve accuracy and reduce overfitting.

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| Python | Core language |
| TensorFlow & Keras | Model building & training |
| CNN (Convolutional Neural Network) | Image classification |
| OpenCV & NumPy | Image preprocessing |
| Streamlit | Web app deployment |
| Jupyter Notebook | Model experimentation |

---

## 📊 Model Performance

- ✅ Applied **data augmentation** (flip, zoom, rotation) to reduce overfitting
- ✅ Used **image resizing & normalization** for preprocessing
- ✅ Saved trained model as `Flower_Recog_Model.h5`
